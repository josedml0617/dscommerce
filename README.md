## Sobre o projeto
Este repositório faz parte do curso da **DevSuperior**, onde estou aprendendo e praticando **Spring Boot** através do projeto **DSCommerce**.

O objetivo é aplicar conceitos de:
- JPA / Hibernate
- Mapeamento objeto-relacional
- Criação de entidades e relacionamentos
- Configuração de banco de dados H2
- Desenvolvimento de API REST
